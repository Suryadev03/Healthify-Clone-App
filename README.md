# 🥗 Healthify Clone App (Gemini API)

An AI-powered **Health & Nutrition Assistant App** inspired by Healthify-style platforms.  
Provides **personalized diet plans, calorie insights, health recommendations, and lifestyle guidance** based on user inputs such as age, weight, goals, and food habits.

---

## 🚀 Features

- 🧠 AI-generated personalized diet plans  
- 🍎 Food & calorie recommendations  
- 🏋️ Workout suggestions based on health goals  
- 📊 BMI & basic health metrics calculation  
- 🩺 Preventive health tips  
- 🤖 Gemini-powered conversational health assistant  
- 📱 Responsive UI (Web-friendly)  

---

## 🛠️ Tech Stack

### Backend
- **Language:** Python  
- **Framework:** FastAPI / Flask  
- **AI Engine:** Gemini (Google Generative AI)  

### Frontend
- **Framework:** Streamlit (or React alternative)  
- **Styling:** Tailwind CSS / CSS  

### AI & Logic
- Prompt-based health reasoning  
- Rule-based + AI hybrid recommendations  

---

## 🔑 Prerequisites

- Python **3.9+**  
- Gemini API Key (Google Generative AI)  
- Node.js (if using React frontend)
